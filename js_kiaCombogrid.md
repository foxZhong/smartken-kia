# Introduction #

本控为复合下拉框，用户可通过输入短码或下拉选择方式快速选定需要的字段，实现快速输入的效果

## 依赖控件 ##
  1. jquery
  1. easyui

## 调用方式 ##

```
     var opts={};
     $("<input />").kiaCrudForm(opts);
```

## 传参方式 ##
**参数必须使用json对象方式传入,如下例**
```
     var opts={};
     opts["colNum"]=4;
```
